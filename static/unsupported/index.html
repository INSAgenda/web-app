Votre navigateur est obsolète

Pour utiliser INSAgenda, faîtes toutes les mises à jour et utilisez un navigateur récent :

  • Brave (recommandé)
  • Firefox
  • Ungoogled Chromium
  • ...ou tous les autres navigateurs majeurs
